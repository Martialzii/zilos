# Changelog

## v0.4.0

- Added activity `tags`
- Added recurring task support with `--recurring` (`none`, `daily`, `weekly`, `monthly`)
- Added `due` command to view due and overdue activities
- Added PayPal setup and subscription stub commands
- Added JSON export support
- Updated docs for versioned access and new features
- Updated package metadata to `0.4.0`

## v0.3.0

- Added `project` support for activities
- Added activity `priority` (`low`, `medium`, `high`)
- Added `stats` command for richer breakdowns by project, user, priority, and category
- Added `export` command to write filtered activities to CSV
- Added versioned alias script `zilos_0_3_0.py`
- Updated package metadata to `0.3.0`
- Updated docs and README for the new version

## v0.2.0

- Added user tracking
- Added average duration reporting by user
- Added `.gitignore`
- Added package metadata
